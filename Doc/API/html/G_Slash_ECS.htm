Namespaces
NamespaceDescription
Slash.ECS
Component-based entity system.
Slash.ECS.Blueprints
Blueprints for creating entities with a specific set of components and initial attribute values.
Slash.ECS.Components
Game components that can be attached to entities.
Slash.ECS.Configurations
Data to create and initialize an entity.
Slash.ECS.Events
Events that occur within the entity system. May contain additional data listeners might be interested in.
Slash.ECS.Inspector.Attributes
Expose component properties to editors.
Slash.ECS.Inspector.Data
Additional property data to show in editors, such as names and descriptions.
Slash.ECS.Inspector.Utils
Utility methods for collecting editor data and initializing objects.
Slash.ECS.Inspector.Validation
Utility classes for validating editor properties.
Slash.ECS.Logging
Logger interface that allows interested listeners to register for log messages.
Slash.ECS.Processes
Allow ticking and queueing timed processes. Good examples are animations, tweens, or "Go to that point, and open the door after."
Slash.ECS.Systems
Game logic systems that are the heart of each game.
